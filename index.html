HTML Cheatsheet

This cheatsheet is the 2nd off world project on the Codecademy Front-End engineering course. I have used this opportunity to get a bit creative and show what I have learned up to this point(13%). The "td" was also selected for specific purposes.

Property

Feature

Description
Display Inline Displays inline element (like <span>). Takes up the width and height of the element only. Will follow the element even if it goes onto anothe line.
Position Relative The element is positioned relative to its normal position, whereas absolute will be positioned to that of its closest ancestor. Left 20px is the same figure but will impact them in different ways although moving the same distance.
Margin Collapse Adds spacing around the element rather than within. With margin collapse, margin-bottom(30px) for the top element will not combine with the second elements margin top(20px) to create 50px. The overall space will be set at margin-bottom.
Padding N/A Adds space within the element and therefore impacts the width of the elment once added. It is the space between the elements content and the border.
